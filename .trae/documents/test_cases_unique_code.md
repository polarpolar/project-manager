# 项目唯一代码功能测试用例

## 测试环境

* 测试URL：<http://localhost:8000/project-manager.html>

* 浏览器：Chrome/Safari/Firefox

* 准备：语雀文档或Excel文件用于导入测试

## 测试用例 1：唯一代码生成规则验证

### 目的

验证生成的项目唯一代码为4位随机字符（数字0-9 + 大写字母A-Z）

### 测试步骤

1. 打开项目管理系统
2. 点击"新建项目"按钮
3. 填写项目信息（项目名称：测试唯一代码1）
4. 选择"洽谈推进中"阶段
5. 保存项目

### 预期结果

* 项目编号格式：`C` + 年月 + 4位随机字符

* 例如：`C2603AB12`

* 末尾4位为数字或大写字母的组合

## 测试用例 4：阶段变更时唯一代码保持不变

### 目的

验证项目阶段变更时，唯一代码不改变

### 测试步骤

1. 创建一个"洽谈推进中"阶段的项目（获得编号如：C2603ABCD）
2. 将项目阶段变更为"已签单·执行中"
3. 填写合同日期

### 预期结果

* 变更后的项目编号：`P` + 合同日期年月 + 唯一代码

* 例如：`P2603ABCD`（末尾ABCD保持不变）

### 实际结果

## 测试用例 5：语雀导入 - 新项目生成唯一代码

### 目的

验证从语雀导入新项目时生成唯一代码

### 测试步骤

1. 准备语雀文档，包含项目表格：

   | 项目名称    | 项目阶段    |
   | ------- | ------- |
   | 语雀测试项目A | 洽谈推进中   |
   | 语雀测试项目B | 已签单·执行中 |
2. 打开导入对话框，切换到"语雀导入"标签
3. 输入文档链接，点击"读取文档"
4. 点击"确认导入"

### 预期结果

* 导入后项目自动生成唯一代码

* 项目编号格式正确

### 实际结果

***

## 测试用例 6：语雀导入 - 识别唯一代码并匹配已有项目

### 测试步骤

1. 先创建一个项目：测试匹配项目，获取唯一代码如：`C2603XY99`
2. 导出项目唯一代码（后续测试用例7会用到）
3. 准备新的语雀文档，包含相同名称但不同的项目信息
4. 导入该文档

### 预期结果

* 系统识别到唯一代码，判定为"更新"而非"新增"

* 更新后唯一代码保持不变

### 实际结果
